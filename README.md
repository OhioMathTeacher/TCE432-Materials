# TCE 432A Course Materials

Student-facing materials for TCE 432A - Adolescent Science Methods II (Spring 2026)

This repository contains PDFs and resources linked from Canvas course pages.

## Structure
- Materials are organized by week
- All PDFs are accessible via raw GitHub URLs
- Updates pushed here automatically update Canvas links

## Usage
Students: Click links from Canvas - files will display directly in your browser.

Instructor: Edit markdown in main TCE432 repo, convert to PDF, copy to this repo, push.
